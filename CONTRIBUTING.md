# Contributing

This repository contains the public specification, documentation, schemas, and site source for CAP.

Contribution rules:

- normative protocol changes belong under `specification/`
- tutorials and explainers belong under `documentation/`
- machine-readable artifacts belong under `schema/`
- `site/` should reuse repo content instead of rewriting it

Editorial constraints:

- do not add product-specific discovery surfaces to CAP core
- do not add implementation details to normative protocol text
- do not introduce capabilities that are not supported by the current CAP source material

Contribution workflow details can expand over time, but these boundary rules should remain stable.
